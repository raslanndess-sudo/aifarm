# YouTube Comment Bot — Техно-Циник

Автоматические провокационные ответы на комментарии под Shorts.

## Быстрый старт

```bash
cd yt-comments
pip install -r requirements.txt
```

Заполни `.env`:
```
ANTHROPIC_API_KEY=sk-ant-...
```

Запуск:
```bash
python bot.py
```

## Что делает

- Каждые 15 минут проверяет последние 5 видео канала
- Генерирует ответы через Claude в стиле «Техно-Циник»
- Публикует с рандомной задержкой 5–20 минут (антибан)
- Лимит 20 ответов/час, режим сна с 01:00 до 07:00
- Стоп-лист: политика, религия, оскорбления

## Файлы

| Файл | Назначение |
|------|------------|
| `bot.py` | Основной скрипт |
| `.env` | API ключи и настройки |
| `token.json` | OAuth токен YouTube |
| `state.json` | Стейт (кому уже ответили) |
| `bot.log` | Логи |

## Настройки антибана (.env)

```
MAX_REPLIES_PER_HOUR=20   # макс ответов в час
JITTER_MIN_MINUTES=5      # мин задержка перед ответом
JITTER_MAX_MINUTES=20     # макс задержка
SLEEP_START_HOUR=1        # начало "сна"
SLEEP_END_HOUR=7          # конец "сна"
```

## Модель

Использует `claude-haiku-4-5` — быстро и дёшево для массовой генерации.
